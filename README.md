# 🚀 SkillMap AI — AI-Powered Career Guidance Platform

React.js | Node.js | PostgreSQL | AI

## 📌 Project Overview

SkillMap AI is an intelligent career development platform that helps students identify skill gaps, generate personalized learning roadmaps, and build ATS-friendly resumes. The platform bridges the gap between academic learning and industry requirements through AI-driven recommendations.

**Key Result:** Provides personalized career guidance, skill analysis, and resume generation to improve student employability.

---

## 🎯 Problem Statement

Many students struggle to understand which skills are required for their target careers and how to build an effective learning path. SkillMap AI helps students identify missing skills, track progress, and create professional resumes for better career readiness.

---

## 🏗️ Project Architecture

Student Profile → Skill Assessment → Skill Gap Analysis → Career Roadmap Generation → Resume Builder → Progress Dashboard

## 🚀 How to Run

Clone Repository:

```bash
git clone https://github.com/SanjaiSaravana/SkillMap-AI
```

Install Dependencies:

```bash
npm install
```

Start Frontend:

```bash
cd student-portal/frontend
npm start
```

Start Backend:

```bash
cd student-portal/backend
npm run dev
```
